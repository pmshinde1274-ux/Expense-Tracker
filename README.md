#  Expense Tracker Pro 

A simple and powerful **Expense Tracker Desktop Application** built using **Python and Tkinter**. This app helps users manage daily expenses, track spending by category, and stay within budget.

---

##  Features

-  Add new expenses (Name, Amount, Category)
-  View total expenses
-  Category-wise expense summary
-  Search expenses
-  Delete selected expense
-  Clear all expenses
-  Save data locally (`data.txt`)
-  Export data to CSV file (`expenses.csv`)
-  Budget warning system

---

##  Tech Stack

- **Python**
- **Tkinter (GUI Library)**
- File Handling (TXT & CSV)
